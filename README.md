# Topmotto

This is a front-end prject created with Angular.

## To Run this project:
* First download this repository
* Open your terminal
* Navigate to the directory on your computer
* Run `npm install`
* Run `npm start`

## Dependencies:
* Bootstrap
* Font Awesome
* Slick Carusel
